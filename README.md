# 💬 Anonymous Chat


## 📖 Overview

Anonymous Chat is a cross-platform mobile application developed using **Flutter** and **Firebase Realtime Database**. The application enables users to communicate anonymously through private chat rooms without creating an account.

It is designed with a clean Material 3 interface, smooth animations, and real-time synchronization to provide a fast and seamless messaging experience.

---

## ✨ Features

- 💬 Real-time anonymous messaging
- 🏠 Create private chat rooms
- 🔑 Join existing chat rooms using a room ID
- 🚀 Instant message synchronization with Firebase
- 🎨 Modern Material 3 user interface
- 🌈 Animated splash screen
- ⚡ Responsive Flutter UI
- 👤 Nickname-based chatting
- 🔥 Firebase Realtime Database integration
- 🚫 No login or registration required

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/anonymous-chat-flutter.git
```

### 2. Navigate to the project

```bash
cd anonymous-chat-flutter
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Configure Firebase

Create a Firebase project and enable:

- Firebase Realtime Database

Then replace the Firebase configuration with your own project credentials.

### 5. Run the application

```bash
flutter run
```

## 📱 How It Works

1. Launch the application.
2. Enter your nickname.
3. Create a new chat room or join an existing room.
4. Start exchanging messages in real time.
5. Leave the room whenever you wish.
